Connect to GitHub

Connect once. After that, every change you save publishes to your live website automatically.

Get your key (one time, 2 minutes):
1. Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
2. Click Generate new token. Name it "Website Admin".
3. Repository access → Only select repositories → pick Fresh-Lanka-Pvt-Ltd
4. Permissions → Repository permissions → set Contents to Read and write
5. Generate, copy the token, paste it below.
Keep it private: the token is stored only in this browser, never on the website. Don't use this on a shared computer. If it leaks, delete it on GitHub and make a new one.
