Product Photos & Logo

Photos live in the assets folder of your GitHub repo.

To add a product photo:
1. Take a clear, square photo of the product.
2. Rename it simply, e.g. okra.jpg (lowercase, no spaces).
3. Upload it to GitHub → assets folder.
4. In the Products tab, set that product's Image to assets/okra.jpg and Save.
To change the logo: upload your new logo to GitHub → assets folder, named exactly emblem.png (replacing the old one). Use a transparent PNG. The whole site updates automatically.
